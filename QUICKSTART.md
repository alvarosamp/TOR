# ⚡ Quick Start - Site TOR

## 🎯 Em 3 Passos, Seu Site Está Online!

### Passo 1: Testar Localmente (2 minutos)

```bash
cd C:\Users\vish8\OneDrive\Documentos\TOR
python -m http.server 8000
```

Abra no navegador: **http://localhost:8000**

✅ Veja seu site funcionando!

---

### Passo 2: Criar Repositório no GitHub (5 minutos)

1. Vá para [https://github.com/new](https://github.com/new)
2. Nome: `tor-website`
3. Selecione **Public**
4. Clique **Create repository**

---

### Passo 3: Fazer Upload (5 minutos)

Copie e execute no terminal:

```bash
cd "C:\Users\vish8\OneDrive\Documentos\TOR"
git remote add origin https://github.com/SEU-USUARIO/tor-website.git
git branch -M main
git push -u origin main
```

*(Substitua `SEU-USUARIO` pelo seu username do GitHub)*

---

### Pronto! 🎉

Seu site estará disponível em:
```
https://SEU-USUARIO.github.io/tor-website
```
